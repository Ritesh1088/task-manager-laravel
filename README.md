# Task Management System

## Tech Stack
- Laravel (Backend)
- Django (Overdue Service)
- React (Frontend)

## Setup

### Laravel
php artisan serve

### Django
python manage.py runserver 8001

### React
npm start

## Features
- Admin & User roles
- Task assignment
- Overdue logic via Django
